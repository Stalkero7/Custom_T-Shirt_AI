# AI T-Shirt Store

This project is a web application that allows users to design T-shirts using AI-generated images and text. Users can also create mockups of their designs on different colored T-shirts.

## Features

*   **AI Image Generation:** Generate images from text prompts using DALL-E.
*   **Text Editor:** Add and style text on your T-shirt design.
*   **Mockup Generator:** Create realistic mockups of your T-shirt designs.
*   **T-Shirt Color Selection:** Choose between different colored T-shirts.

## Tech Stack

*   **Frontend:** React, Fabric.js
*   **Backend:** Node.js, Express
*   **AI:** DALL-E
